# Talking Companion

An app that announces places nearby. 

## How to Build

1. Download and install latest version of Xcode with supporting Swift (e.g. https://imzdl.com/osx.php).
2. Clone the project.
3. Launch "Talking Companion.xcodeproj".
4. Run the project (Cmd + R).
