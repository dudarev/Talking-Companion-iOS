# Talking Companion

An app that announces places nearby. 

## Compatibility

Swift 1.2 and Xcode 6.3, 6.4b.

## How to Build

1. Install [Carthage](https://github.com/Carthage/Carthage#installing-carthage). 
2. Run `carthage update`.
